# Log Your Habit

A console-based habit tracker to log daily habits, view monthly dashboards, streaks, weekly summaries, and pending habits.  
Built with **Python standard library only**. No external dependencies.

---

## Features

- Monthly calendar with today highlighted
- Habit completion counts and % for the month
- Longest streaks per habit (across months)
- Weekly summary (last 7 days)
- Top habit (most consistent)
- Quick log for today
- Log for any day
- Reset data for a specific month

---

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/yourusername/log-your-habit.git
```

2. Navigate into the folder:
```bash
cd log-your-habit
```

3. Run the program:
```bash
python habit_tracker.py
```

---

## Requirements

- Python 3.x (standard library only)
